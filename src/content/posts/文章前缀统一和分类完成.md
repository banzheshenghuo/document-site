## 更新内容

### 文章前缀规范

#### 精读文章
```bash
[精读] 文章标题.md
[解读] 文章标题.md
[深度] 文章标题.md
```

#### 收藏文章
```bash
[收藏] 文章标题.md
[归档] 文章标题.md
[书签] 文章标题.md
```

### 首页 Tab 分类规则

文档站首页会根据文件名前缀自动分类：

| Tab | 前缀 | 说明 |
|-----|------|------|
| 全部 | 无 | 显示所有文章 |
| 📚 精读 | `[精读]`、`[解读]`、`[深度]` | 只显示精读文章 |
| ⭐ 收藏 | `[收藏]`、`[归档]`、`[书签]` | 只显示收藏文章 |
| 📅 今天 | 当天创建的文章 | 只显示 24 小时内的文章 |

## 使用建议

### 1. 创建新文章

在创建新文章时，确保文件名包含对应的前缀：

**精读文章**：
```bash
# 使用解读文章触发词
输出文件名：[精读] 新文章标题.md

# 在文章头部添加分类
---
title: 文章标题
type: 精读
---
```

**收藏文章**：
```bash
# 使用收藏文章触发词
输出文件名：[收藏] 新文章标题.md

# 在文章头部添加分类
---
title: 文章标题
type: 收藏
---
```

### 2. 验证分类

创建文章后，可以在首页点击不同 Tab 查看该文章是否正确显示：

- **全部 Tab**：应该能看到所有文章
- **精读 Tab**：应该只能看到精读文章
- **收藏 Tab**：应该只能看到收藏文章
- **今天 Tab**：应该只能看到当天创建的文章

### 3. 搜索功能

文章搜索功能会同时搜索文章标题和前缀，因此：
- 搜索"精读"可以找到所有精读文章
- 搜索"收藏"可以找到所有收藏文章
