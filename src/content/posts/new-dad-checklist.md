# 准爸爸准备清单

恭喜你即将成为爸爸！这份清单帮你提前准备产后护理和新生儿相关事项。

---

## 📋 出生证明办理

### 准备材料

- [ ] 父母双方身份证原件及复印件
- [ ] 父母双方户口簿原件及复印件
- [ ] 结婚证原件及复印件
- [ ] 医院出具的《出生医学证明首次签发登记表》
- [ ] 填写《出生医学证明自填单》（需父母双方签字）

### 办理流程

1. **出生后24小时内**：医院会发放《出生医学证明首次签发登记表》
2. **填写信息**：仔细核对新生儿姓名、父母信息（一旦签发，改名将很麻烦）
3. **提交材料**：在出院前向医院提交材料
4. **领取证明**：医院办理后发放《出生医学证明》

### 注意事项

- 新生儿姓名需提前确定，避免临时更改
- 确保所有信息准确无误
- 妥善保管，遗失补办麻烦

## 👶 新生儿护理准备

### 必备物品

- [ ] 尿不湿（新生儿NB号2-3包）
- [ ] 湿纸巾（婴儿专用）
- [ ] 婴儿衣物（52-59码，纯棉材质）
- [ ] 包被/襁褓
- [ ] 婴儿奶瓶（2-3个）
- [ ] 配方奶粉（若非全母乳喂养）
- [ ] 婴儿洗护用品（沐浴露、润肤露）
- [ ] 婴儿指甲剪
- [ ] 体温计
- [ ] 婴儿监护器（可选）

### 护理知识

- **脐带护理**：每天用碘伏消毒1-2次，保持干燥
- **黄疸监测**：出生后每天观察皮肤、眼白颜色，必要时测黄疸值
- **喂养**：新生儿每2-3小时喂一次，按需喂养
- **睡眠安全**：仰卧睡，床上不要放枕头、被子等杂物
- **洗澡**：水温37-38℃，每天或隔天一次

## 🏥 产后护理（妈妈）

### 医院待产包

- [ ] 身份证、医保卡、产检病历
- [ ] 待产服、哺乳内衣（2-3件）
- [ ] 产妇卫生巾（加长夜用）
- [ ] 一次性内裤（1-2包）
- [ ] 抽纸、湿纸巾
- [ ] 充电器、保温杯、吸管杯
- [ ] 毛巾、脸盆
- [ ] 宝宝换洗衣物
- [ ] 拍照设备（记录重要时刻）

### 产后恢复（妈妈）

- **饮食**：均衡营养，多喝水，补充蛋白质
- **休息**：保证充足睡眠，让家人分担照顾宝宝
- **恶露观察**：颜色由鲜红→粉红→白色，异常及时就医
- **伤口护理**：侧切/剖腹产伤口保持清洁干燥
- **情绪支持**：产后情绪波动正常，多陪伴倾听

## 📋 证件办理

### 医保相关

- [ ] 新生儿医保登记（出生后3个月内办理）
- [ ] 生育津贴申请（通过公司或社保局）

### 户口登记

- [ ] 新生儿户口登记（出生后1个月内）
- [ ] 所需材料：出生证明、父母身份证、户口簿、结婚证

### 疫苗接种

- [ ] 出生时：乙肝疫苗（第一针）、卡介针
- [ ] 建立预防接种证
- [ ] 了解当地接种时间表

---

## 💡 提前准备建议

### 孕期

- [ ] 参加产前课程，学习育儿知识
- [ ] 提前选择医院（建档）
- [ ] 了解医院停车、探视规定
- [ ] 准备好紧急联系人名单

### 临产前

- [ ] 检查待产包是否齐全
- [ ] 确认医院路线和备用路线
- [ ] 提前安排好工作和休假
- [ ] 与家人分工明确（谁负责开车、谁照顾大宝等）

### 产后

- [ ] 安排产后访视（42天复查）
- [ ] 准备好月嫂或家人支持
- [ ] 了解当地产后恢复资源（通乳、康复等）
