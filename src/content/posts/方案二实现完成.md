## 实现内容

### 1. DocCard 组件优化

**修改的文件**：
- `/Users/zq/Documents/ob/ai-document/src/components/DocCard.astro`

**关键变化**：
1. **移除了前缀分离逻辑**：不再单独提取和显示前缀标签
2. **直接显示完整标题**：`title` 字段包含完整的前缀和标题
3. **添加左边框颜色**：为标题添加左边框，根据文章类型显示不同颜色

### 2. 样式方案

**精读文章**（`[精读]`、`[解读]`、`[深度]`）：
- 左边框颜色：蓝色 `#3b82f6`
- 背景渐变：`linear-gradient(90deg, #3b82f6 0%, #e0e7ff 10%)`
- 左边框宽度：4px
- 圆角：0.5rem
- 内边距：0.5rem

**收藏文章**（`[收藏]`、`[归档]`、`[书签]`）：
- 左边框颜色：橙色 `#f59e0b`
- 背景渐变：`linear-gradient(90deg, #f59e0b 0%, #ffbf00 10%)`
- 左边框宽度：4px
- 圆角：0.5rem
- 内边距：0.5rem

**无前缀文章**：
- 默认样式，无特殊颜色
- 正常边框

### 3. 样式代码

```css
/* 精读文章：蓝色左边框 */
.doc-card--read .doc-card-title {
  border-left-color: #3b82f6;
  background: linear-gradient(90deg, #3b82f6 0%, #e0e7ff 10%);
}

/* 收藏文章：橙色左边框 */
.doc-card--collect .doc-card-title {
  border-left-color: #f59e0b;
  background: linear-gradient(90deg, #f59e0b 0%, #ffbf00 10%);
}

/* 标题通用样式 */
.doc-card-title {
  font-size: 1.25rem;
  font-weight: 600;
  margin-bottom: 0.5rem;
  color: var(--color-text);
  line-height: 1.4;
  padding: 0.5rem 0.5rem;
  border-radius: 0.5rem;
  border-left: 4px solid transparent;
}
```
